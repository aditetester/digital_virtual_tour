# Gecko Digital Virtual Tours

A professional virtual tour application built with React Native and Expo.

## Features
- Immersive 360Â° virtual tours
- Real-time analytics and notifications
- Seamless booking and meeting scheduling
- Interactive social features (likes, comments)

## Technical Stack
- **Frontend**: React Native, Expo (Bare Workflow), Expo Router
- **Backend**: Hono (Standalone Node Server)
- **API**: tRPC
- **Database**: SurrealDB

## Getting Started

### Prerequisites
- Node.js
- npm or bun

### Installation
1. Install dependencies:
   ```bash
   npm install --force
   ```

2. Start the backend:
   ```bash
   npm run backend
   ```

3. Start the app:
   ```bash
   npm run start
   ```

## Native Development
- To run on Android: `npm run android`
- To run on iOS: `npm run ios`
